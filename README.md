# 🚗 AutoDrive Notebook

An AI-powered automotive assistant inspired by NotebookLM, built with **Gemini Canvas** and powered by the **Google Gemini API**.

Upload your vehicle documents—including owner's manuals, service manuals, insurance policies, and warranty information—and chat with them using natural language. The app also includes a vehicle health dashboard and a service timeline to help manage maintenance history.

## Features

* 🤖 AI chat with automotive documents
* 📄 Support for manuals, insurance, and warranty files
* 🔑 Bring your own Gemini API key (BYOK)
* 🚘 Vehicle health dashboard
* 📅 Service timeline for maintenance tracking

## Tech Stack

* **Built with:** Gemini Canvas
* **Frontend:** HTML, Tailwind CSS, JavaScript
* **AI:** Google Gemini API
* **Deployment:** Vercel

**Live Demo:** *https://autodrive-notebook.vercel.app/*
